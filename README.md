# K-Means-Clustering

(Algoritmo de Agrupamiento No Supervisado — Implementación Manual en Python)

Este proyecto implementa desde cero el algoritmo K-Means en Python, con el objetivo de agrupar puntos o entidades en clusters según su similitud espacial o numérica.
El enfoque principal consiste en simular una ciudad donde se ubican casas y hospitales, utilizando el método de distancia euclidiana para asignar cada casa al hospital más cercano.

El algoritmo realiza iteraciones hasta alcanzar la convergencia, actualizando los centroides (hospitales) con base en la media de sus asignaciones.
Incluye métricas de desempeño (distancia promedio, mínima, máxima, desviación estándar) y una representación visual tipo mapa ASCII para analizar la distribución de los clusters.

En resumen, este proyecto demuestra de manera práctica cómo el algoritmo K-Means puede optimizar la ubicación de recursos o segmentar datos de manera eficiente, sin necesidad de librerías externas como scikit-learn.
